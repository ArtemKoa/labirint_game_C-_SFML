Данный проект предназначен для обучения и повторения уже полученных навыков программирования на C++. Работа включала в себя создание алгоритма случайного создания лабиринта, в котором всегда найдется минимум 1 путь выхода. Далее создавалось окно для игры, чтобы появилась возможность передвижения по лабиринту. Следующим пунктом стало создание классов для каждой части игры(создание лабиринта, танк и т.д.), а также двусвязный список для того, чтобы генерировать неопределенное количество танков. Была организована физика передвижения танков, повороты и увеличение, уменьшение их скорости, базы на которых спавнятся вражеские танки со своими определенными в классах характеристиками.
