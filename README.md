Данный проект предназначен для обучения и повторениям уже полученных навыков программирования на C++. Работа включала в себя создание алгоритма случайного создания лабиринта, в котором всегда найдет минимум 1 путь выхода. Дален создавалось окно для игры, чтобы появилась возможность передвижения по лабиринту. Следующим пунктом стало создание классов для каждой части игры(создание лабиринта, танк и т.д.), а также двусвязный список для того чтобы генерировать неопределенное количество танков. Была организована физики передвижения танков, повороты и увеличение, уменьшение их скорости.
